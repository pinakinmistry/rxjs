What is AngularJS?

  • Superheroic JavaScript MVW Framework.
  • What HTML would have been, had it been designed for building web-apps.
  • A client side framework for building dynamic web apps.
  • Extends HTML with new syntax/construct called directives (ng-some-directive).
  • Sprinkle some ng-* and see Angular's magic.

HTML++

How it all started?

  • Google Feedback Project in 2009.
  • 18,000+ lines of client side code base in 6 months.
  • Misko Hevery challenged to rewrite it in 2 weeks.
  • He failed.
  • He succeeded in 3 weeks.
  • Code base came down to 1500 lines.
  • Yay, Superheroic it is!

ng-father = Misko Hevery

Why Angular?

  • Removes lot of complexity/boilerplate code.
  • Make common tasks trivial and difficult tasks possible.
  • Easy to learn.
  • Opinionated.
  • Driven by Google/Community.
  • Most in demand/popular client side framework.

Core Features:

  • Directives to extend HTML
  • 2 way data binding
  • MVC, MVVM, MVW, MV*
  • Client side templating
  • Client side routing
  • Dependency Injection
  • Modular, Reusable, Testable Components

Angular components:

  • Directives (ng-app, ng-controller, ng-repeat, ng-show, ng-hide, ng-view, ...)
  • Module (angular.module())
  • Controller (ng-controller)
  • Model (ng-model)
  • Scope ($scope)
  • Dependency Injection of $scope
  • Angular expression ({{}})
  • View
  • Filter (| filter)
